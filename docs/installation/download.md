# 获取 openSUSE

> 你可以直接访问 [software.opensuse.org](https://software.opensuse.org/) 获取相关信息。

## 如何获取 openSUSE 发行版

如果您在中国，我们建议您使用一个镜像站点进行下载。您可以直接右键复制链接地址直接进行下载。
